# k-Nearest Neighbor
dado uma nova instância não rotulada, o processo de inferência se dá ao comparar a distância entre ela e todas as instâncias na base de dados e, assim, rotulá-la.

depende do valor de k.

Possiveis hiperparâmetros:

| Identifier | Class name | args | distance fn |
|------|-------|------|------|
| euclidiana  |  |  | 
| manhattan   |  |  | 
| chebyshev    |  |  | 
| minkowski    |  |  | 



ver algoritmo no slide depois

complexidade:
    O(n): para calcular as distâncias e encontrar o vizinho mais próximo.
    O(nk): constante, dado por nk (numero de vizinhos * complexidade da entrada)

na classificação, podemos calcular a moda das classes 
na regressão, média ou mediana

propostas de seleção de um k ótimo: 
- Validar o modelo com testes de diferentes k. 
- Heurísticas.
- Análise de Viés-Variância
