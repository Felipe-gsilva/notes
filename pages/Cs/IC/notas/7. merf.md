# Representação de Cena pelo MERF (Merf's Scene Representation)

O merf representa um volume atraves de:
- um campo de densidades $r \in \mathbb{R}_+$ 
- cores difusas $c_d \in \mathbb{R}^3$
- vetores de características $f \in \mathbb{R}^K$

Esses valores são renderizados usando o modelo de shading do SNeRG.

Esse campo é parametrizado de maneira diferente do Nerf de Mildenhall, que utilizava uma MLP para o fazer. Esse projeto utiliza-se de grids distintos:
- um voxel grid de baixa resolução ($L \times L \times L$)
- 3 grids de alta resolução, um para cada um dos planos ($P_x = yz$, $P_y = xz$ e $P_z = xy$) - $R \times R$. 

Cada elemento desses grids armazena um vetor com 4 + K canais 

com $L = 512$, $R = 2048$ e $K = 4$ por default.

O campo continuo de vetores C é a soma dos vetores 3D trilinearmente interpolados e os 2D bilinearmente interpolados, dado por:

$$\tau(x, y, z) = V(x,y,z) + P_x(x,y,z) + P_y(x,y,z) + P_z(x,y,z)$$

todo resultado dessa operação pode ser expandido para uma densidade , cor difusa e característica f dependente da visualização, os quais são, depois, submetidos a não linearidade a seguir:

$$ \tau = exp(\tilde{\tau} ) \ , \ c_d = \sigma(\tilde{c}_d \ , \ f = \sigma(\tilde{f})$$

onde $\sigma$ é função sigmoid logistica comum.
